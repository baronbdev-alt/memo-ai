# AI Quiz Maker

This project allows you to generate quizzes (True/False or Multiple Choice) using AI.

===========================================================================================

## Setup Instructions

1. create venv 
    == python -m venv venv

2. activate venv
    ==  .\venv\Scripts\activate

3. to download the requirements:
    ==  pip install -r requirements.txt

4. run AI Quiz Maker
    ==  python .\main.py

===========================================================================================

If an error occurs, delete the venv folder and start again from Step 1.
